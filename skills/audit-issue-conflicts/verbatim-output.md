# Output Evidence Contract (verbatim-output rule)

When this skill emits a conflict finding, verdict, or ranked report, cite
evidence verbatim rather than re-summarizing — quoting the conflicting issue
text is cheaper than paraphrasing and keeps the audit auditable:

IMPORTANT: For each conflict you report:
1. State your verdict: Yes / No / Partial (is this a genuine conflict?)
2. Provide a VERBATIM quote from each issue's text (exact text, in quotes) that establishes the conflict.
3. If you cannot quote specific text from both sides, your verdict is automatically No (or Partial if only one side is clearly stated).

Do not assert a conflict without evidence. "These issues seem to overlap" is not evidence.
