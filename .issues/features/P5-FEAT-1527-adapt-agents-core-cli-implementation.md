## Generated TOML Format

```toml
# generated by ll-adapt-agents-for-codex
name = "codebase-analyzer"
description = "Use this agent when you need to understand HOW code works..."
model = "sonnet"

developer_instructions = """
Use this agent when you need to understand HOW code works - tracing implementation details...
[full body after frontmatter ---]
"""
```

Fields: `name` (string), `description` (first non-empty line of block scalar, ≤80 chars via `_extract_short_desc`), `model` (string), `developer_instructions` (multiline string, full body after `---`). F-strings only, no `tomli_w`.

## Test Helper Pattern

```python
def _make_agent(tmp_path: Path, name: str, description: str = "...", model: str = "sonnet",
                body: str = "Agent instructions.") -> Path:
    agents_dir = tmp_path / "agents"
    agents_dir.mkdir(parents=True, exist_ok=True)
    agent_md = agents_dir / f"{name}.md"
    agent_md.write_text(
        f"---\nname: {name}\ndescription: |\n  {description}\nmodel: {model}\n"
        f"tools: [\"Read\"]\n---\n\n{body}"
    )
    return agent_md
```
